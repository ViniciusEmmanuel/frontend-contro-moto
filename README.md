## Propósito

- Criar uma interface para CRUD simples de gastos com motos (gasolina e manutenção).

## Desenvolvido

- [ReactJs](https://pt-br.reactjs.org/)

## Pré requisitos

- npm ou yarn
- node >= 12
- backend [https://github.com/ViniciusEmmanuel/ApiControleMoto]()

## Start

- yarn ou npm i
- yarn ou npm start
